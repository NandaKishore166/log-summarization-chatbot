# Log Summarization Chatbot 🤖

This project is an AI-based log summarization system with a chatbot interface.

## Features
- Log preprocessing
- Event extraction
- AI-based summarization
- ChatGPT-style UI

## Technologies
- Python
- Flask
- NLP (Transformers)

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python chatbot_ui.py

3. Open browser:
   http://127.0.0.1:5000/

- show errors
- warnings
- summary
